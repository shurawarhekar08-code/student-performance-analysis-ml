# Student Performance Analysis & Prediction System

## Overview
This project analyzes student academic data and predicts exam scores based on
study hours and attendance using Machine Learning.

## Dataset
- Source: Kaggle (Student Performance Factors dataset)
- Features used:
  - Hours_Studied
  - Attendance
  - Exam_Score

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Steps Performed
- Exploratory Data Analysis (EDA)
- Data Visualization
- Model Training using Linear Regression
- Model Evaluation
- Prediction for new student data

## How to Run
1. Install dependencies  
   `pip install -r requirements.txt`
2. Run the script  
   `python student_performance.py`
